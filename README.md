## 이 화면의 배포가 오래된 이유

이 저장소의 배포 주소는 `https://vokim2503.github.io/First-Lotto/`이며 예전 버전입니다.  
현재 앱 주소는 `https://vokim2503.github.io/`입니다.

이 저장소의 기존 파일과 배포 기록은 이전 개발 과정을 확인할 때만 사용하세요.
